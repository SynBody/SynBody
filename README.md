# SynBody: Synthetic Dataset with Layered Human Models for 3D Human Perception and Modeling

<div align="center">
    <img style="width:100%" src="https://synbody.github.io/static/teaser.png">
    <strong>SynBody</strong> is a large-scale synthetic dataset with massive number of subjects and high-quality annotations. It supports various research topics, including human mesh recovery and novel view synthesis for human (Human NeRF).
</div>

----

Project page: [SynBody](https://synbody.github.io)

Paper is released at [arxiv](https://arxiv.org/pdf/2303.17368.pdf)

Codes & details will be released soon.
